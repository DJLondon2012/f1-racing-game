# 🏎️ F1 Racing Pro - Ultimate Racing Game

## Overview
A high-performance 3D racing game built with Three.js featuring realistic physics, stunning graphics, and multiple game modes. Race against AI opponents, beat your best times, or compete in championship seasons.

## 🎮 Features

### Game Modes
- **Grand Prix**: Full 5-lap race against AI drivers
- **Time Trial**: Single lap challenge to beat your best time
- **Quick Race**: Fast 3-lap race
- **Championship**: Multi-round season competition

### Realistic Gameplay
- Advanced physics engine with smooth steering and drift mechanics
- Real-time fuel management system
- Vehicle damage tracking
- Tire wear simulation
- Dynamic RPM and gear display
- Position tracking against opponents

### Stunning Graphics
- High-quality 3D models with detailed cars
- Advanced lighting system with shadows
- Post-processing effects (bloom, tone mapping)
- Procedurally generated track with curves and elevation
- Realistic road textures with tire marks
- Dynamic weather-ready environment

### User Interface
- Intuitive main menu with mode selection
- Real-time HUD displaying speed, RPM, fuel, damage
- Mini-map showing track and competitor positions
- Garage system for vehicle customization (expandable)
- Detailed race completion screen with stats
- Touch controls for mobile devices
- Keyboard support for desktop

### Performance Optimization
- Optimized renderer settings for smooth gameplay
- Efficient shadows and lighting
- Dynamic LOD management
- Mobile-friendly touch controls
- Responsive design for all screen sizes

## 🎮 Controls

### Desktop
- **Left Arrow** - Steer Left
- **Right Arrow** - Steer Right

### Mobile
- **Left Button** - Steer Left (bottom left)
- **Right Button** - Steer Right (bottom right)

## 🚀 How to Play

1. **Start Game**: Click "PLAY RACE" from main menu
2. **Select Mode**: Choose your desired race mode
3. **Race**: Use arrow keys or touch buttons to steer
4. **Complete Race**: Finish all laps to see your results
5. **View Stats**: Check final position, time, and damage

## 📊 Game Statistics

- **Speed**: Current velocity in KPH
- **Gear**: Automatic gear based on speed
- **RPM**: Engine revolutions per minute
- **Fuel**: Decreases as you drive
- **Damage**: Accumulates during race
- **Tire Wear**: Increases with distance
- **Position**: Your current rank vs opponents
- **Lap**: Current lap progress
